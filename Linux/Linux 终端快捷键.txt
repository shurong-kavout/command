
Linux 终端快捷键

# 移动光标：
Ctrl + a：移到行首
Ctrl + e：移到行尾
Esc + b：往回(左)移动一个单词
Esc + f：往后(右)移动一个单词
 
# 删除字符：
Ctrl + h：删除光标左方位置的字符
Ctrl + d：删除光标右方位置的字符（注意：当前命令行没有任何字符时，会注销系统或结束终端）
 
# 删除单词：
Esc + d：由光标位置开始，删除单词，直到该单词结束
Ctrl + w：由光标位置开始，往左删除单词

# 删除行部分：
Ctrl + k：由光标所在位置开始，删除右方所有的字符，直到该行结束
Ctrl + u：由光标所在位置开始，删除左方所有的字符，直到该行开始

# 删除整行：
Ctrl + a && Ctrl + k  或  Ctrl + e  && Ctrl + u 或 Ctrl + k && Ctrl + u 

# 清除屏幕：
Ctrl + l
 
# 复原操作：
Ctrl + _：回复之前的状态撤销操作
 
# 粘贴：
Ctrl + y：把之前删除的字符或字符串，贴到光标所在位置
 
# 重复执行操作动作：
Esc + 操作次数
 
# 查找历史命令
Ctrl + p：显示当前命令的上一条历史命令
Ctrl + n：显示当前命令的下一条历史命令
 
# 搜索历史命令
Ctrl + r
