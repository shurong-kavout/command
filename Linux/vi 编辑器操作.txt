1. 打开文件
$ vi  文件      # 该模式是命令模式

2. 尾行模式操作
$ :q            # 该模式是“尾行模式”
$ :w            # 保存已经修改的文档
$ :wq			# 保存并退出
$ :q!			# 强行退出，不保存修改
$ :/内容/		# 查找指定内容，n 向上查找，N 向下查找
			    # 查找到文档的最后，自动返回文档的开头
$ :set nu		# 给文件每行内容设置编号
$ :set nonu		# 取消编号设置
$ :s/bin/Bin/   # 替换当前行首次出现的 bin 为 Bin
$ :s/bin/Bin/g	# 替换当前行所有的 bin 为 Bin
$ :%s/bin/Bin/g # 替换所有行的 bin 内容为 Bin
$ :n			# 光标跳转到 n 行位置
$ /内容			# 查找指定内容


3. 命令模式光标移动
# 字符移动：上、下、左、右 键盘
     h(左)   j(下)   k(上)   l(右)

# 单词移动
    w ： 到达下个单词的首字母
    b ： 到达本单词或上个单词的首字母
    e :  到达本单词或下个单词的尾字母

# 行级移动
    $  :  到达行尾
    0  :  到达行首

# 段落级移动
    {  :  到达段落尾部
    }  :  到达段落尾部

# 屏级移动(只在当前屏有效，不会滚动屏幕)
    H  :  到达可见屏幕头部
    L  :  到达可见屏幕尾部

# 文档级移动
    gg :  到达文档首行
    G  :  到达文档末行
    1G :  到达文档首行
    nG :  到达第 n 行


4. 命令模式删除文档信息

# 字符级删除    x
# 通用删除      d+光标移动距离
$ d+$ 从当前位置删除到行尾
$ d+0 从单前位置删除到行首
$ d+w 删除单词(包括单词后边的空格)

$ d+} 删除到段落尾部(尾部空行有保留)
$ d+{ 删除到段落的首部(首部上边的空行也被删除，只删除一行空白行)

$ d+L  删除至屏幕尾部, 删除包括当前行
$ d+H  删除至屏幕首部，删除包括当前行

$ d+G  删除至文档尾部，删除包括当前行
$ d+1G 删除至文档首部，删除包括当前行
$ d+nG 删除至文档的第n行，删除包括当期行和第n行

$ dd   删除当前行
$ n+dd  从当前行向下删除n行


5. 命令模式复制并粘贴
$ yy    复制单行
$ n+yy  复制n行
$ y+光标移动距离    可以进行灵活复制  [参考删除]
$ p     粘贴(光标所在位置的后边开始粘贴，[光标所在行的后一行])

6. 删除的内容，也可以被粘贴，删除的内容被放到缓存里边，也可以提供粘贴服务


7. 在“命令模式”对操作进行撤销
$ u        # 撤销，向上会递归撤销以往所有操作
$ r+字符   # 单个字符替换
$ .        # 重复执行上个命令
$ J        # 合并两行

8. 将命名执行结果导入正在编辑的内容（尾行模式）
$ :r !date
$ :r !ls -li
